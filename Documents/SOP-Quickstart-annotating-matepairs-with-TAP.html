SOP - Annotating amplicon mate pairs

TAP CWL Workflow

Authors: Andreas

Version: 0.9.1

Audience:

This document is intended for users who are familiar with docker and amplicon data analysis.

Requirements:

  • text editor
  • mate pair sequence files  
    (e.g.     sequences_R1.fastq and sequences_R2.fastq)
  • docker
  • wget

Known limitations: 

  • memory > 8 GB (depends on data input size and classification algorithm)

Introduction:

This SOP describes the steps necessary to run the TAP (prokaryote) pipeline.

SOP:

  1. git clone https://github.com/MG-RAST/amplicon.git
  2. Change into cloned repository:
    cd amplicon
  3. Download reference databases:
    ./setup.sh

    or from within the container

    docker run -v `pwd`:/amplicon -ti --rm --entrypoint bash  --workdir /amplicon mgrast/amplicon:latest setup.sh

  1. Change into the newly created Data directory:

    cd Data

    This will be your working directory. Copy or move your sequence files into this directory and     rename them to R1.fastq.gz and R2.fastq.gz . For example:
  1. cp your_path/ your_sequence_files.R1.fastq.gz > R1.fastq.gz
  2. cp your_path/ your_sequence_files.R2.fastq.gz > R2.fastq.gz
  1. docker run -ti --rm -v `pwd`:/Data mgrast/tap:latest prok

Notes:

  • To get help:

    docker run -ti --rm -v `pwd`:/Data mgrast/tap:latest help
  • Run any command in container:

    docker run -ti --rm -v `pwd`:/Data mgrast/tap:latest     COMMAND
